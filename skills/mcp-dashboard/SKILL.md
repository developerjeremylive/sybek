# MCP Dashboard

Este skill proporciona una UI web para gestionar servidores MCP.

## Uso

Simplemente pregunta:
- "Muestra el dashboard de MCP"
- "Abre la UI de MCP"
- "Ver servidores MCP"

## Características

- Lista de todos los servidores MCP configurados
- Estado de cada servidor
- Información de herramientas disponibles
- Botones para habilitar/deshabilitar servidores
